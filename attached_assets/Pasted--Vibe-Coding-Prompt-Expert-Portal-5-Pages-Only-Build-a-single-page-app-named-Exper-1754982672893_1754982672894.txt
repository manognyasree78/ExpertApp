# Vibe Coding Prompt — Expert Portal (5 Pages Only)

**Build a single-page app named “Expert Knowledge Portal”.**
Use a light theme (white cards, soft shadows). Top header + left sidebar layout. All data is static and persisted to `localStorage`.

## 1) Shell & Routing

* Create a **PortalShell** layout with:

  * **Header (top bar):** left title “Expert Knowledge Portal”; right side shows `Hi, {name}` and a circular avatar with initials.

    * **Name source:** `localStorage.auth.user.name`. If missing, default to `"Dr. Sarah Johnson"`. Also store `localStorage.auth.user.email` for consistency. Initials from name.
    * Avatar dropdown: **Profile Settings**, **Logout** (Logout clears `localStorage.auth` and navigates to `/portal/dashboard` with default name again).
  * **Sidebar (sticky):** items (with icons) → routes:

    * Dashboard → `/portal/dashboard`
    * Contribute Knowledge → `/portal/contribute`
    * My Knowledge Status → `/portal/status`
    * Earnings → `/portal/earnings`
    * Profile Settings → `/portal/profile`
    * Support → `/portal/support` (simple contact card)
* **Router:**

  * `/portal` → redirect to `/portal/dashboard`
  * `/portal/dashboard` → Dashboard page
  * `/portal/contribute` → Contribute Knowledge
  * `/portal/status` → My Knowledge Status & Feedback
  * `/portal/earnings` → Earnings & Analytics
  * `/portal/profile` → Profile Settings
  * `/portal/support` → Support (small card with `support@ottobon.in`)
* All navigation is client-side; show an active state on the current sidebar item.
